# TerraE

**TerraE** (*terrae*, Latin: "of the earth") is a large-scale hydrological land model that pays homage to [ModelE](https://github.com/nasa-giss/modelE), NASA GISS's climate model.

## Python Prototype

The main development is in the **terrae_py** package—a single-column soil hydrology prototype with explicit and implicit Richards equation solvers.

→ **[terrae_py/README.md](terrae_py/README.md)** — Installation, quick start, and technical details
